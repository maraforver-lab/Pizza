# Patch 323 homepage hero asset review

Patch 323 approved two local homepage hero assets:

- No people.
- No faces.
- No hands or arms.
- No silhouettes.
- No human reflections or human shadows.

- No embedded text.
- No logos.
- No watermarks.
- No floating UI panels or app mockups.

The desktop image is used as a decorative, text-safe background for the homepage hero and invitation artwork. The mobile image is used as meaningful hero imagery with concise alt text.
